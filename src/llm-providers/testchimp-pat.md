# TestChimp PAT Provider

Configure the TestChimp PAT provider by setting:

- `TESTCHIMP_USER_AUTH_KEY` – required
- `TESTCHIMP_USER_MAIL` – required
- `TESTCHIMP_BACKEND_URL` – optional (defaults to `https://featureservice.testchimp.io`)

Use this option when you prefer a personal access token instead of project-level credentials.
